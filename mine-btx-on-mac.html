Mac setup checklist

  1. Create a BTX wallet address before running the installer.
  2. Open Terminal on the Mac.
  3. Run the Mac preflight command from the command builder.
  4. Run the install command with your BTX address and a worker label.
  5. Watch your address dashboard on the BTX Start home page.

When Mac mining makes sense

Mac mining is useful for learning BTX mining with hardware you already own. Max and Ultra machines may be more interesting for sustained testing, but GPU rental can be a better option when you need faster hashrate.
