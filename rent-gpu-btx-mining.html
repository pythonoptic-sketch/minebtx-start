How rental GPU mining works

Choose a rental GPU, launch the instance, run the BTX Start installer, and use the same personal BTX payout address you use locally. The page dashboard groups workers by payout address, so rented rigs and local rigs can be tracked together.

Referral disclosure

BTX Start may earn Vast.ai referral credit when someone signs up through the BTX Start referral link and buys credits. This does not change the user's BTX payout address, private keys, pool dashboard, or mining rewards.
