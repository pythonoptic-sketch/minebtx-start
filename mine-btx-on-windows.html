Windows BTX mining

Mine BTX on Windows with WSL2.

Windows miners should use Ubuntu under WSL2, verify NVIDIA GPU access with nvidia-smi, then run the BTX Start preflight and install commands inside Ubuntu.

Open command builder Full Windows notes

Windows setup checklist

  1. Install the current NVIDIA driver in Windows.
  2. Install Ubuntu under WSL2 with wsl --install -d Ubuntu.
  3. Open Ubuntu and confirm nvidia-smi works.
  4. Paste your BTX payout address on the home page.
  5. Run the generated preflight and install commands inside Ubuntu.

Good Windows candidates

Windows desktops with NVIDIA GPUs are usually better candidates than small laptops. If you do not own a strong GPU, BTX Start also links to Vast.ai rental GPU pricing for faster experiments.
